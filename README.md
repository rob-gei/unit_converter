A Console based unit converter in python. Its pretty self explaining since every step is print out in the console. 

It can convert the most common units:
  -Kilogramms
  -Gramms
  -Milligramms
  -Pounds
  
  -Foot
  -Miles
  -Inch
  -Kilometer
  -Meter
  -Centimeter
  -Millimeter
  
  -Gallon's 
  -Liter
  -Milliliter

I made it to learn more about dictionarys in python.
